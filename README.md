# Vacas e Ovelhas

Projeto desenvolvido para o Concurso Agrinho 2026.

## Recursos

- HTML5 semântico
- CSS3
- JavaScript
- Modo escuro
- Aumento e diminuição de fonte
- Responsividade
- Imagens com créditos

## Objetivo

Apresentar a importância das vacas e das ovelhas para a vida no campo e para todos.
